How to run a HTML file from browser

Ref: https://www.webucator.com/blog/2016/06/launch-files-browser-visual-studio-code/ Windows Method: --------------- Press Ctrl+Shift+P to open the Command Palette. Type “ctr” and select the Configure Task Runner command to open tasks.json. Delete the current placeholder contents. Enter the following: { "version": "0.1.0", "command": "Chrome", "windows": { "command": "C:\\Program Files (x86)\\Google\\Chrome\\Application\\chrome.exe" }, "args": ["${file}"] } To test it, open an HTML file in Visual Studio Code and press Ctrl+Shift+B. It should open in Chrome. Alternate Option: http://127.0.0.1:5500/02.HTML5-tags.html Click on Live-Server in the right corner of the interface and open the above url in browser
